# (C# 코딩) LoginScreen

## 개요

* C# 프로그래밍 학습

* 1줄 소개: 사용자로부터 아이디와 비밀번호를 입력받아 로그인 여부를 판단하는 프로그램

* 사용한 플랫폼:

* C#, .NET Windows Forms, Visual Studio, GitHub

* 사용한 컨트롤:

* Label, TextBox, Button

* 사용한 기술과 구현한 기능:

* Visual Studio를 이용하여 UI 디자인

* string 클래스를 이용한 사용자 입력 데이터 처리

* 조건문(if문)을 이용한 로그인 판단

* MessageBox를 이용한 로그인 결과 출력

* 수업 중에 배우고 사용했던 클래스들 관련된 설명

* TextBox: 사용자 입력을 받는 컨트롤

* Button: 클릭 이벤트를 통해 기능 실행

* MessageBox: 사용자에게 결과를 보여주는 창

* 실습 중에 구현한 기능들 설명

* 아이디와 비밀번호 입력 기능 구현

* Placeholder 기능 구현 (Enter, Leave 이벤트 활용)

* 비밀번호 입력 시 ● 표시 기능 구현

* 로그인 성공/실패 판단 기능 구현

## 실행 화면 (과제1)

* 과제1 코드의 실행 스크린샷

![과제1 실행화면](1차과제2.png,제목없음.png)

* 과제 내용

* Label(표시), TextBox(입력), Button(로그인)을 적절히 배치합니다.

* 아이디와 비밀번호 입력창에 안내 문구(Placeholder)를 구현합니다.

* 로그인 버튼 클릭 시 입력값을 비교하여 로그인 성공/실패를 판단합니다.

* 결과를 MessageBox로 출력합니다.

* 구현 내용과 기능 설명

* 프로그램 실행 시 입력창에 "아이디", "패스워드" 안내 문구가 표시된다.

* 입력창 클릭 시 안내 문구가 사라지고 사용자가 입력할 수 있다.

* 비밀번호 입력 시 입력값이 ● 형태로 표시된다.

* 로그인 버튼 클릭 시 미리 설정된 아이디와 비밀번호와 비교한다.

* 아이디와 비밀번호가 모두 일치하면 로그인 성공 메시지가 출력된다.

* 하나라도 틀리면 로그인 실패 메시지가 출력된다.
